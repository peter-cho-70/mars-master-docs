01 Master — 아이템 추가하기 Master

큐시트 표에서 아이템을 더블클릭하거나 ✏️ 버튼을 누르면 편집창이 열립니다. 툴바의 + 아이템 추가를 누르면 새 아이템을 만들며 곧바로 같은 창이 열립니다. 아래는 실제로 "리포트 - 국회 예산안 처리 상황" 아이템 하나를 처음부터 끝까지 채운 화면입니다.

  • 기본 정보 — 타입(바꾸면 그 타입의 기본 전환효과·카메라로 자동 맞춰짐)·No·아이템명·담당/기자· 시간(mm:ss)·전환In/Out. 지금 값이 그 타입의 기본값과 다르면 바로 아래 회색 안내 줄에 "지금 값이 기본과 다름(의도한 예외인지 확인)"이 떠서, 실수로 다른 전환을 골랐는지 바로 알 수 있습니다.
  • 카메라 — 타입에 정의된 카메라 후보 중 실제 쓸 카메라를 체크합니다(예시는 CAM2).
  • 송출 서버 / 영상 클립 — + 클립 추가로 클립 행을 만들고, 📁 찾기로 실제 파일을 고르거나 파일명을 직접 입력합니다. ▶ 미리보기로 맞는 파일인지 그 자리에서 확인할 수 있습니다.
  • Videohub 라우터 전환 — 출력/입력을 고르면 이 아이템이 온에어될 때 그대로 적용됩니다. TAKE를 누르면 저장·방송 여부와 무관하게 즉시 그 크로스포인트로 전환해 확인할 수 있습니다(이 예시는 실장비 미연결이라 "Videohub 미연결" 안내만 보입니다).
  • 자막 — 하단자막(자막바) 한 줄과, + CG 자막 추가로 여러 개의 CG(이름자막·티커 등 타입별)를 추가합니다. 자세한 내용은 06장을 참고하세요.
  • 기타 — 앵커멘트(프롬프터 원고, 진행 중 프롬프터 패널에 그대로 표시)와 비고. 글자 수/줄 수 안내(권장: 한 줄 20자 이내·6줄 이내)가 실시간으로 표시됩니다.
  • 진행 미리보기 — 저장하기 전, 지금 설정대로라면 실제 온에어에서 어떤 순서로 컷/디졸브가 일어날지 미리 볼 수 있습니다(예시: "앵커: CAM2 > 컷 > 서버: item1.mov"). 하단 저장을 눌러야 실제로 반영됩니다.
시간(mm:ss)은 클립을 실제 파일로 연결하면 자동으로 갱신됩니다 클립을 여러 개 붙인 아이템은 실제 파일 길이를 다 아는 클립들의 합으로 "시간"이 자동 갱신됩니다 — 직접 고쳐도 되지만, 클립을 다시 연결하면 실측값으로 다시 덮어씁니다.

02 Master — 중계(LIVE) 시퀀스 채우기 Master

타입을 LIVE(중계)로 바꾸면 "중계 시퀀스" 섹션이 새로 나타납니다. 질문 수와 단계별 소스를 지정한 뒤 ↻ 질문 수로 단계 자동 생성을 누르면, 아래처럼 실제 진행 단계 목록이 만들어집니다.

  • 질문 수 — 기사(MARS)에서 넘어오지만, 방송 중 실제 진행 상황에 맞춰 언제든 여기서 조정할 수 있습니다.
  • 앵커 갈라치기 / 중계 풀샷(또는 기자) / 서버 갈라치기 소스 — 비워두면 아이템의 "기타 소스"를 그대로 씁니다. 갈라치기 소스는 해당 구도를 미리 구성해둔 vMix DVE/Multi-View 인풋을 스위처 소스로 지정하세요.
  • ↻ 질문 수로 단계 자동 생성 — 기본 반복 패턴(앵커 갈라치기→중계 풀샷→서버 갈라치기, 질문 수만큼 반복 후 마무리)을 만듭니다. 그 뒤 아래 단계 목록에서 + 단계 추가로 특정 질문만 단계를 더 넣거나, ×로 빼거나, ▲▼로 순서를 바꿀 수 있습니다.
출연(스튜디오 게스트)도 같은 화면을 씁니다 MARS의 '출'(출연) 코드로 매칭된 아이템도 같은 중계 시퀀스 엔진을 쓰되, 라벨이 "앵커 갈라치기(DVE2)/ 기자/기자 서버 갈라치기(DVE1)"로 바뀌고 마지막 단계가 지정한 Full Shot으로 마무리됩니다.

03 MARS — 아이템 추가하기 MARS

MARS의 아이템 편집창은 Master보다 기사 작성에 더 초점이 맞춰져 있고, 자막의 IN/OUT을 영상을 보면서 직접 찍는 기능이 편집창 안에 바로 들어있는 것이 가장 큰 특징입니다. 큐시트 표에서 아이템을 더블클릭하면 열립니다.

  • 형식 / No — 타(타이틀류)·완(완제 리포트)·단(단신)·출(출연)·중(중계)· NT(녹취)·CM·특(오프닝/클로징 등)·서(서버단독영상)·복(복대자막) 중에서 고릅니다. Master와 코드 체계가 다르지만 자동으로 대응되어 전달됩니다.
  • 아이템명 / 제목 — "제목"은 큐시트 표의 '제목' 열로, 기사를 매칭하면 자동 채워지지만 직접 입력·수정도 가능합니다.
  • 담당/기자 · 시간(mm:ss) · 전환In/Out — 이 형식의 기본 전환효과와 다르면 바로 아래 안내 줄에 표시됩니다(Master와 동일한 규칙).
  • 카메라 샷 / 영상 클립 — 큐시트 표의 "샷 지정"과 같은 항목입니다. "영상 개수"에 숫자를 넣으면 클립 슬롯을 한 번에 늘리거나 줄일 수 있고, "찾기"로 실제 파일을 연결하면 파일명이 저장됩니다.
  • 자막 — 인/아웃 설정(왼쪽) — + 자막 추가로 자막을 만들고, 타입(제목자막/하단자막/이름자막 등)과 문구를 입력합니다. "수동 지정" 체크박스는 기본이 해제(수동)이며, 켜면 영상 길이 기준 자동 계산으로 바뀝니다.
  • 영상 재생 + 타임라인(오른쪽) — 클립을 연결하면 바로 이 자리에서 실제 영상이 재생됩니다. ▶IN/■OUT으로 지금 재생 위치를 찍고, 아래 타임라인 막대(초록색 "이름자막" 구간)로 결과를 바로 확인할 수 있습니다. 자세한 사용법은 05장을 참고하세요.
Master의 편집창과 무엇이 다른가요? MARS는 vMix에 직접 연결하지 않으므로 Videohub 라우터 TAKE 같은 "실제로 지금 장비를 건드리는" 버튼이 없습니다. 대신 자막 IN/OUT을 실제 영상과 함께 찍는 기능이 편집창 자체에 내장돼 있어, 기사를 쓰는 단계에서부터 타이밍까지 한 번에 끝낼 수 있습니다. 여기서 만든 내용은 "방송 진행"을 누르면 Master로 그대로 전달됩니다.

04 MARS — 자막(CG) 설정 상세 MARS

아이템 편집창 말고도, 오른쪽 자막조회 탭에서 큐시트 전체의 자막을 한 번에 관리할 수 있습니다. 아이템을 선택하면 이 탭에 그 아이템의 자막 목록이 나타납니다.

① 오버레이별 자막 미리보기 (8슬롯)

슬롯기본 이름보통 쓰이는 곳
Ovl1제목자막리포트 상단 헤드라인, 인용구
Ovl2하단자막단신 복대, 일반 하단 설명
Ovl3이름자막기자/인터뷰이 이름·소속
Ovl4시간자막시각/날짜 표시
Ovl5위치자막촬영 장소
Ovl6코너자막날씨/증권 등 코너 로고
Ovl7긴급자막속보/긴급 문구
Ovl8티커하단 흐르는 줄자막

각 슬롯의 이름은 클릭해서 자유롭게 바꿀 수 있습니다(이 프로그램에만 저장, 실제 vMix 오버레이 번호 배정은 Master 설정에서 별도로 합니다). 이 미리보기는 문구만 보여줄 뿐 실제 송출은 아닙니다.

② 단신 자동 자막 타이밍 기본값

여기 적은 IN(초)/끝나기(초) 전 OUT은 공식일 뿐입니다 — 실제로 어느 자막에 적용할지는 각 자막 아래 "위 기본값으로 이 자막 자동 IN/OUT 계산" 체크박스가 정합니다.

기본값은 "수동"입니다 — 체크해야만 자동 계산이 적용됩니다 예전에는 단신 자막이 기본적으로 체크된 상태(자동)라, 아무 것도 지정하지 않은 자막까지 방송 중 저절로 자동 전송되는 문제가 있었습니다. 지금은 기본이 해제(수동 CG로만 동작)이고, 명시적으로 체크를 켜야만 이 공식대로 영상 길이 기준 IN/OUT을 자동 계산합니다. ☑ 지금 큐시트의 모든 단신에 위 공식(체크박스) 켜기 버튼을 누르면 큐시트 안의 모든 단신을 한 번에 자동으로 켤 수 있습니다(이미 손으로 지정해둔 값은 지우지 않고 남겨둡니다).

③ 자막 목록

선택한 아이템의 자막이 하나씩 카드로 나열됩니다. 각 카드에서 문구를 고치고, 체크박스로 자동/수동을 정하고, 삭제로 지울 수 있습니다. 카드 아래 "🎬 타이밍" 버튼(또는 편집창의 IN/OUT 버튼)으로 실제 영상을 보며 정확한 지점을 찍는 방법은 다음 장을 참고하세요.

05 영상 보면서 자막 IN/OUT점 찍기 MARS

자막이 화면에 나타나고(IN) 사라지는(OUT) 정확한 시점은, 실제 영상을 재생하며 눈으로 보고 찍는 것이 가장 정확합니다. MARS는 이 기능을 두 곳에서 제공하며, 결과는 완전히 같은 데이터를 공유합니다.

  1. 아이템 편집창에서 바로 — 03장 스크린샷처럼, 클립을 연결하면 편집창 안에 영상과 타임라인이 바로 나타납니다. 자막을 추가한 뒤 영상을 재생하고, ▶IN/■OUT을 원하는 순간에 누르면 됩니다.
  2. 전용 새 창에서(자막조회 탭) — 자막조회 탭의 ▶ 자막 인/아웃 설정 (새 창)을 누르면, 화면을 크게 쓰면서 리포트처럼 자막이 여러 개인 아이템을 하나의 재생 흐름으로 순서대로 찍을 수 있는 전용 창이 열립니다 — 화면 구성·조작법 등 자세한 내용은 자막(CG) 인/아웃 설정 가이드를 참고하세요.

공통 사용법

  1. 클립을 실제 파일로 연결합니다(영상이 없으면 재생 영역에 "연결된 영상 클립이 없습니다"만 표시됩니다).
  2. 영상을 재생합니다. 브라우저가 자동재생을 막으면 영상을 한 번 클릭하거나 재생 버튼을 누르세요.
  3. 자막이 나타나야 할 순간에 ▶IN을 누릅니다.
  4. 사라져야 할 순간에 ■OUT을 누릅니다 — 이 순간 "수동 지정" 체크박스가 자동으로 켜집니다(직접 찍은 값을 쓰겠다는 뜻).
  5. 타임라인의 막대 좌우 손잡이를 드래그하면 IN/OUT을 각각 미세 조정할 수 있고, 타임라인을 클릭하면 그 위치로 재생 지점이 바로 이동합니다.
  6. 잘못 찍었으면 지우기로 취소하고 다시 찍으면 됩니다 — 지우면 기본 상태(수동, 타이밍 없음)로 돌아갑니다.
"저장"을 눌러야 반영됩니다 편집창에서 찍은 자막 타이밍은 다른 수정 내용과 마찬가지로 저장을 눌러야 실제 큐시트에 반영됩니다. 저장하지 않고 창을 닫으면 방금 찍은 타이밍도 함께 사라집니다.

06 Master와 MARS, 자막 설정의 차이

MARSMaster
자막을 어디서 만드나 기사 작성 후 자동 매칭, 또는 아이템 편집창/자막조회 탭에서 직접 추가 MARS에서 전달받은 그대로 사용, 또는 아이템 편집창의 "자막" 섹션에서 직접 추가/수정
IN/OUT 타이밍 영상을 실제로 보면서 찍거나(수동), 단신은 체크박스로 자동 계산 가능 MARS가 보낸 타이밍을 그대로 쓰거나(있으면), 없으면(CG In 딜레이 미지정) 방송 중 CG 큐에서 ON 버튼으로 수동 전송
실제 송출 안 함 — 화면 미리보기용(vMix에 직접 연결하지 않음) 실제로 vMix GT 타이틀 또는 브라우저 오버레이로 송출
